# Worldweaver.engine
Welcome to WorldWeaver.Engine. A work in progress Procedural tabletop world generation engine that generates entire planets with interactive towns and maps and even street views. This tool is being made to help people in the tabletop RPG and worldbuilding community as a way to create worlds for your campaigns or start your own campaigns on the site with potential online features in the future.

# Features
- Procedural 3D planets (seed-based)
- Heightmaps, biomes, And 
- Google Maps–style zoomable world maps
- Procedural towns and streets
- NPC simulation
- DM controls
- Map saving and loading and exporting/importing
- multiplayer markers and collaboration (potential still undecided)
- Notes and progression trackers
